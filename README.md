# Mental Wealth Academy

A research-driven ai learning system focused on improving cyber-psychology, and pragmatic governance systems using blockchain technology.

## Overview

Mental Wealth Academy is designed to help proliferate education, by sharing digital tools for real-world communities, helping them transition into a digital future education that makes decision-making more transparent, crystal-clear, and accessible for everyone. The platform provides opportunities to take back sequestered knowledge, and build a better future, for all of us, together.

## Key Features

- **Educational Workshops**: Comprehensive resources for community learning and development
- **Library**: Organized library of community resources and assets
- **Messaging**: Community discussion boards for knowledge sharing and collaboration
- **Quests**: Interactive learning paths and community challenges

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript, Solidity.
- **Deployment**: Vercel
